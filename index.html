Directory Placement

This index.html file must be at the same level as the TeslaCam folder which can be placed directly on the root of the USB drive.

Example:

Root Folder
|- index.html
|- TeslaCam(USB Drive Top Level Folder)
   |- Saved Clips
   |- Sentry Videos
   |- ...
